# LeetCode Solutions in Java

This repository contains my personal Java solutions to LeetCode problems, organized by topic.

## Folder Structure

- `Arrays/` – Problems like Two Sum, Best Time to Buy and Sell Stock, etc.
- `Strings/` – Problems involving substrings, palindromes, etc.
- `LinkedList/` – Classic problems like Merge K Sorted Lists, Swap Nodes in Pairs.
- `Trees/` – DFS, BFS, traversal problems.
- `Recursion/` – Backtracking & recursive logic like Generate Parentheses.
